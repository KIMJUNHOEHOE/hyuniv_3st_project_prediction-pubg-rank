# hyuniv_3st_project_prediction pubg rank
 
캐글에서 주최하는 대회 중 PUBG(게임 배틀그라운드) 승률 예측하는 프로젝트 진행
- train set : 약 450만개, test set : 약 200만개, number of features : 29개

- 데이터 통계 분석 : EDA를 활용한 데이터 분석, 상관관계 분석, 주성분 분석, 두 표본 집단 분석 등일 이용
- 데이터 이해를 통한 변환
- 머신러닝 및 딥러닝을 활용한 배틀그라운드 승률 예측
- 결과 상위 75%(1500팀 참여)

* 사용 언어 및 패키지
- Python tensorflow(DNN)
- Python stats model
- Python sklearn(Randomforest, xgboost, Linear Regresssion 등)
- Python matplotlib
- Python Seaborn
